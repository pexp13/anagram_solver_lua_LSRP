# anagram_solver_lua_LSRP
Lua-скрипт для SAMP (LSRP) на MoonLoader (+ImGui). Представляет из себя автоматизированный решатель анаграмм (генерирует перестановки букв, фильтрует реальные слова, извлекает текст и так далее), позволяет также быстро отправлять нужные слова в чат. Разработан в 2024-2025 как личный инструмент.
